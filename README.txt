# Real Time Programing - Assignment 1 - CST8244 - Algonquin College - Computer Engineering - Computing Science

-  Design and build a building entry controller system simulation with a testing harness.
-  Implement a two-door system with card scanners, a scale for weighing persons, and a human guard-operated switch for door control.
-  Design a state machine for the system, detailing inputs, outputs, state transitions, and provide a diagram.
-  Develop three programs: des_display for system status display, des_controller for state machine operation and data maintenance, and des_inputs for simulating input events.
-  Test the programs to ensure proper function and adherence to specifications.
